# 🎮 Game Success Prediction Tool

A Java console-based application that predicts the potential success of a game using a rule-based scoring system.

## 🚀 Features
- Takes user input for:
  - Genre (Action / Puzzle / RPG)
  - Platform (PC / Console / Mobile)
  - Critic Score (0–10)
  - Difficulty (Easy / Medium / Hard)
- Calculates a score based on predefined logic
- Outputs prediction:
  - High Success Potential
  - Medium Success Potential
  - Low Success Potential

## 🧠 How It Works
The application assigns weighted scores to each input parameter:
- Genre influences popularity
- Platform affects reach
- Critic score reflects quality
- Difficulty impacts accessibility

The total score determines the predicted success level.

- Java
- OOP concepts
- Conditional logic & decision-making
